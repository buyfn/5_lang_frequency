# Frequency Analysis of Words

You give this script txt file - it shows 10 most frequent words in it.

Example:
```
python lang_frequency kapitanskaya_dochka.txt

Most frequent words in the file:
и: 1302
я: 823
в: 769
не: 643
на: 480
что: 479
с: 456
меня: 321
он: 308
мне: 277
```
